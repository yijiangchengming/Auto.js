# 简介

Auto.js (Pro)是Android上基于JavaScript的面向编程学习与效率提升的代码开发平台。

官网: https://pro.autojs.org

官方文档: https://pro.autojs.org/docs

## 开源说明

本仓库为Auto.js 4.1的开源代码，从Auto.js Pro开始不再开源。
